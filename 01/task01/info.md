## Пояснение:
```
hello_world — Возвращает строку «Привет, мир!».
```
```
cars — Возвращает список машин через запятую. Список вынесен в глобальную область.
```
```
cats — Возвращает случайную породу кошек. Список пород также вынесен в глобальную область.
```
```
get_time/now — Возвращает текущее время в формате YYYY-MM-DD HH:MM:SS.
```
```
get_time/future — Возвращает время через один час.
```
```
get_random_word — Выбирает случайное слово из книги «Война и мир», загруженной при старте приложения.
```
```
counter — Считает количество посещений страницы с помощью атрибута функции.
```
